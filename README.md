# SGReady
SG Ready control of appliances via MQTT
Subscribe to SG Ready topics according to the following four states (A,B):

Inhibit mode 1:0
Normal mode 0:0
Low price mode 0:1
Over capacity mode 1:1 


Topic: SGReady 
Message: Inhibit, Normal, LowPrice,OverCapacity
